GitHub + React.js Guide

Learn how GitHub works with React.js projects step-by-step.

1. Install Git

Download and install Git on your computer.

git --version

2. Create GitHub Account

Create a free GitHub account and verify your email.

3. Install VS Code

VS Code is a code editor used for React.js development.

4. Create React App

Create a React project using Vite.

npm create vite@latest my-app -- --template react

5. Start Project

Install dependencies and run your project.

cd my-app
npm install
npm run dev

6. Initialize Git

Start Git tracking inside your project.

git init

7. Commit Code

Save your project changes.

git add .
git commit -m "First Commit"

8. Push to GitHub

Upload your project to GitHub.

git remote add origin YOUR_REPO_LINK
git push -u origin main

Important Git Commands

Command Purpose
git status Check changed files
git add . Add all files
git commit -m "message" Save changes
git push Upload code
git pull Download latest code
git clone URL Copy repository

What is Git?

Git is a version control system used to track code changes.

What is GitHub?

GitHub is a cloud platform for storing and sharing code.

Why React?

React helps create fast and modern websites.
